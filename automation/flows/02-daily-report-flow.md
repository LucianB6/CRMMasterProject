# Flux 02 - Raport zilnic (agent)

## Scop
Validarea completarii raportului zilnic, autosave si submit.

## Preconditii
- Agent autentificat.
- Formularul zilnic este disponibil.

## Pasi
1. Deschide formularul zilnic.
2. Completeaza campurile numerice.
3. Verifica autosave (draft).
4. Apasa Submit.

## Rezultat asteptat
- Draftul este salvat automat.
- Submit finalizeaza raportul.
- Managerul vede datele actualizate.
