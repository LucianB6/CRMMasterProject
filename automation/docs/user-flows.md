# Fluxuri utilizator (draft)

## 1. Autentificare
- Utilizator primeste credentiale (admin/manager creeaza cont).
- Utilizator se autentifica.
- Sistemul redirectioneaza catre Dashboard.

## 2. Raport zilnic (agent)
- Agent deschide formularul zilnic.
- Completeaza campuri numerice.
- Se salveaza automat ca draft.
- Agent apasa Submit la final.
- Dashboardul managerului se actualizeaza.

## 3. Dashboard & KPI (manager)
- Manager selecteaza echipa/agent.
- Vizualizeaza KPI-uri si trenduri.
- Verifica rapoarte zilnice.

## 4. Auto-finalizare la 23:59
- Daca exista draft, se trimite automat.
- Daca nu exista draft, se completeaza cu zero.
